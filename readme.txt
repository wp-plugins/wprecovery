== Description ==

=Welcome to WPRecovery = – Your WordPress Recovery Manager that will get your WordPress Site back up and running, if it ever stops working correctly.
WPRecovery is revolutionary in both ease-of-use and power! It is designed keeping the end user in mind and takes automated backup tool to next level. If your WordPress site is down, this plugin can get your site back and running in seconds!

What makes WPRecovery different: Our premises behind WP Recovery plugin is that in an event something goes wrong, end user need a simple, one click feature within the WordPress dashboard to restore site to last working state. WPRecovery challenges the traditional approach of focusing on backup and bells and whistles attached to them. Instead, WPRecovery focuses on getting the ‘Restore’ job done, and done with ease.

WPRecovery is released as Beta at this point – we encourage users to share feedback with us and help us developing this plugin further. We also welcome support from the WordPress developer community

Released under GPL for WordPress community.


== Installation ==

1. Download WPRecovery

2. Go to your Dashboard and locate Plugins tab. Click on Upload Plugin and specify the path of downloaded plugin in the text box.

3. Click on Activate Plugin



== Frequently Asked Questions ==

= Does it backups up both files and database =

Yes, WPRecovery backs up both WordPress files and database.
