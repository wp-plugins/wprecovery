Recover and Restore Your WordPress Site If It Ever Stop Working correctly. 

== Description ==

WPRecovery makes recovering your WordPress website a breeze! It makes backups quietly in the background when you make major changes to your website. And if anything goes wrong, a single click will restore your site to the last backup made. 


== Installation ==

1. Download WPRecovery

2. Go to your Dashboard and locate Plugins tab. Click on Upload Plugin and specify the path of downloaded plugin in the text box.

3. Click on Activate Plugin



== Frequently Asked Questions ==

= Does it backups up both files and database =

Yes, WPRecovery backs up both files and database.
